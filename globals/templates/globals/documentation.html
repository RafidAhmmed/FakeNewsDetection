Fake News Detection Documentation

This documentation provides an overview of the Fake News Detection system and its usage. The project is designed to detect whether a news article is fake or real based on the input provided.

Overview

The Fake News Detection project uses machine learning models to classify news articles as "fake" or "real". The system analyzes the textual content of a news article and predicts its validity. We have used a large language model such as BERT for training and prediction.

How It Works

The process for fake news detection involves the following steps:

  1. Input: User submits a news article.
  2. Preprocessing: The text is cleaned, tokenized, and transformed for model input.
  3. Model: A pre-trained model (e.g., BERT) is used to predict the class of the news as 'Fake' or 'Real'.
  4. Output: The result (Fake/Real) is displayed to the user.

Technology Stack

  • Backend: Django Framework
  • Machine Learning: spaCy, BERT
  • Database: PostgreSQL (or other, depending on your setup)
  • Frontend: HTML, CSS, JavaScript
  • Deployment: Docker (optional)

Installation Guide

Follow these steps to set up the project on your local machine:

# Clone the repository
git clone https://github.com/username/FakeNewsDetection.git

# Navigate into the project directory
cd FakeNewsDetection

# Install required dependencies
pip install -r requirements.txt

# Run migrations
python manage.py migrate

# Start the Django server
python manage.py runserver

API Endpoints

The following API endpoints are available for interacting with the Fake News Detection system:

  • GET / - Home page that displays news items.
  • POST /predict - Accepts input data (news text) and returns the prediction (Fake/Real).

Usage

To use the Fake News Detection system:

  1. Visit the home page and input the news article text in the provided form.
  2. Click "Submit" to analyze the news.
  3. The system will display the result: either "Fake" or "Real".
