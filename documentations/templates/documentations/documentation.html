Fake News Detection Documentation

This documentation provides a brief understanding of the Fake News Detection system and its usage. The project is designed to detect whether a news article is fake or real based on the input provided.


Overview

The Fake News Detection project uses machine learning models to classify news articles as "fake" or "real". The system analyzes the textual content of a news article and predicts its validity. We have used a large language model (LLM) such as BERT for training and prediction.


Also their is scrapping approach for more recent news that have not been covered during the mdoel training process.

How It Works


The Training Model Approach works as follows:

  1. Input: User submits a news article.
  2. Preprocessing: The text is cleaned, tokenized, and transformed for model input.
  3. Model: A pre-trained model (e.g., BERT) is used to predict the class of the news as 'Fake' or 'Real'.
  4. Output: The result (Fake/Real) is displayed to the user.


The Scrapping Approach works as follows:

  1. Input: User submits a news article.
  2. Scrapping: Scrapping is done by using the Google Custom Search Engine API
  3. Output: The scrapped result is displayed to the user with context similarity score using Spacy model en_core_web_lg model which facilitates named entity recognition (NER).

Technology Stack

  • Backend: Django Framework
  • Machine Learning: spaCy, BERT
  • Database: SQLite3 with ORM
  • Frontend: HTML, CSS, JavaScript
  • Deployment: Docker

Installation Guide

Follow these steps to set up the project on your local machine:

# Clone the repository
git clone https://github.com/SakibSibly/FakeNewsDetection.git

# Navigate into the project directory
cd FakeNewsDetection

# Install required dependencies
pip install -r requirements.txt

# Run migrations and apply them
python manage.py makemigrations
python manage.py migrate

# Start the Django server
python manage.py runserver

# Create a superuser for admin access
python manage.py createsuperuser

API Endpoints

The following API endpoints are available for interacting with the Fake News Detection system:

  • GET / - JSON response from the server which includes newsfeed and public details of user profile.

Usage

To use the Fake News Detection system:

  1. Open your account
  2. Visit the home page and input the news article text in the provided form.
  3. Click "Analyze" to analyze the news.
  4. The system will display the result: either "Fake" or "Real".
  5. Or user the scrapping approach
